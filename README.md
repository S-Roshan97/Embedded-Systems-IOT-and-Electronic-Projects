# ⚡ Embedded Systems, IoT & Electronic Projects

A collection of my **Embedded Systems, IoT, and Electronics Projects** focused on sensors, automation, robotics, and real-world hardware applications.

This repository demonstrates practical implementations using Arduino and sensor modules, along with circuit designs and code examples useful for beginners, hobbyists, and electronics enthusiasts.

---

# 🚀 Projects Included

## 🌡️ 1. Weather & Environment Monitoring System
- **Sensors:** BMP280 / BMx280  
- **Measures:**
  - Temperature (°C)
  - Atmospheric Pressure (Pa / hPa)
  - Altitude (m)
- **Display:**
  - Serial Monitor
  - LCD Display  

---

## 🌈 2. Color Detection System
- **Sensor:** TCS34725 RGB Color Sensor  
- **Measures:**
  - RGB values
  - Lux (light intensity)
  - Color temperature (Kelvin)
- **Output:**
  - Serial Monitor
  - LED response system  

---

## ☀️ 3. UV Index Monitoring System
- **Sensor:** SI1145 UV Sensor  
- **Displays UV Index Levels:**
  - Low
  - Medium
  - High
  - Extreme
- **Output:**
  - 16×2 I2C LCD Display  

---

## 📏 4. Ultrasonic Distance Meter
- **Sensor:** HC-SR04 / NewPing Library  
- **Function:**
  - Measures distance in centimeters
- **Display:**
  - LCD Display
  - Serial Monitor  

---

## 🚨 5. Ultrasonic Alarm System
- **Sensor:** HC-SR04  
- **Function:**
  - Triggers buzzer when object is within range
- **Use Case:**
  - Security system
  - Parking assistance  

---

## 🤖 6. Servo Automation System
- **Control:** Ultrasonic sensor + Servo motor  
- **Applications:**
  - Automatic gate system
  - Smart door system  

---

## 🎮 7. IR Remote Control System
- **Module:** IR Receiver + Servo Motor  
- **Features:**
  - Control servo using IR remote
  - Supports multiple angle positions (0°, 90°, 180°)  

---

## ❤️ 8. Heart Rate Monitor
- **Sensor:** Analog pulse sensor  
- **Function:**
  - Calculates BPM (Beats Per Minute)
- **Display:**
  - LCD with animation  

---

## ✋ 9. Gesture Control System
- **Sensor:** APDS9960  
- **Detects Gestures:**
  - UP
  - DOWN
  - LEFT
  - RIGHT
- **Output:**
  - LCD display  

---

## 🔐 10. Touch-Based Door Lock System
- **Input:** Touch sensor  
- **Output:** Servo motor + LCD  
- **Function:**
  - Lock / Unlock mechanism  

---

## 💡 11. LED Effects Projects
- Random LED pattern generator

- LED animation system

## 🔊 12. Sensor Alarm Systems

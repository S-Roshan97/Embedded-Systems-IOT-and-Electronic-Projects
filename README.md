# Embedded-Systems-IOT-and-Electronic-Projects
A collection of All My Embedded Systems , IOT and Electronic Projects focused on sensors, electronics, robotics, and IoT applications. This repository showcases practical implementations, circuit designs, and code examples useful for beginners and hobbyists building real-world embedded systems.

🚀 Projects Included
🌡️ 1. Weather & Environment Monitoring System
-Sensors: BMP280 / BMx280
-Measures:
-Temperature (°C)
-Atmospheric Pressure (Pa / hPa)
-Altitude (m)
-Display: Serial Monitor / LCD

🌈 2. Color Detection System
-Sensor: TCS34725 RGB Color Sensor
-Measures:
-RGB values
-Lux (light intensity)
-Color temperature (Kelvin)
-Output: Serial Monitor / LED response

☀️ 3. UV Index Monitoring System
-Sensor: SI1145 UV Sensor
-Displays UV Index levels:
-Low
-Medium
-High
-Extreme
-Output: 16×2 I2C LCD

📏 4. Ultrasonic Distance Meter
-Sensor: HC-SR04 / NewPing library
-Measures distance in centimeters
-Display: LCD + Serial Monitor

🚨 5. Ultrasonic Alarm System
-Sensor: HC-SR04
-Output: Buzzer alarm when object is close
-Use case: Security / parking system

#6. Servo Automation System
-Control servo motor using ultrasonic sensor
-Applications:
-Automatic gate
-Smart door system

🎮 7. IR Remote Control System
-Module: IR Receiver + Servo Motor
-Control servo using TV remote
-Supports multiple commands (0°, 90°, 180°)

❤️ 8. Heart Rate Monitor
-Analog pulse sensor
-Calculates BPM (Beats Per Minute)
-Display: LCD with custom animation

✋ 9. Gesture Control System
-Sensor: APDS9960
-Detects:
-UP, DOWN, LEFT, RIGHT gestures
-Output: LCD display

🔐 10. Touch-Based Door Lock System
-Input: Touch sensor
-Output: Servo motor + LCD
-Function:
Lock / Unlock system

💡 11. LED Effects Projects

-Includes:

Random LED pattern generator
Knight Rider scanner effect
LED animation system

🔊 12. Sensor Alarm Systems
Digital sensor + buzzer
Ultrasonic proximity alarm
Security alert systems
